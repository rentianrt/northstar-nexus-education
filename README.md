# 北辰智汇 · Northstar Nexus Education

北辰智汇公司宣传网站。网站以“在 AI 时代，为成长建立坐标”为核心叙事，将 AI + 科研、美国升学与职业发展组织为一条连续、可验证的成长路径。

## 网站内容

- 品牌定位与“方向 → 项目 → 证据 → 叙事 → 机会”成长模型
- AI + 科研、美国升学、职业发展三大服务矩阵
- 研究方向、导师网络与阶段性交付方式
- 邮件、电话与微信咨询入口

## 本地开发

需要 Node.js `>=22.13.0`。

```bash
npm install
npm run dev
npm run build
```

## 技术结构

- Next.js / React
- vinext / Vite
- Cloudflare Workers 兼容构建
- 响应式单页布局，无外部运行时数据依赖
